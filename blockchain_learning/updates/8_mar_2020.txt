Learning till 8th Mar 2021:
    Blockchain:
        - Blockchain is special database which is secure, anonymous, unanimous, time-stapmed, Immutable, Distributed.
        - A type of DLT(Distributed Ledger Technology)
        - Uses hash to map the blocks and PKI for verification
    
    Bitcoin concepts:
        - Mining and miner
        - Proof of work
        - Incentive for mining winner and solving confilict for parallel winners
        - Merkle Tree
        - Payment verification
        - Network

    Possible attacks on bitcoin in future(with next gen techs):
        - Race Attack
        - Finney Attack
        - The 51% Attack
